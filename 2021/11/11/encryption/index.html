      # тЇЋтљЉтіат»єсђЂтЈїтљЉтіат»є

      # тЅЇУеђ

      тюеС╣ІтЅЇуџёТќЄуФаWebжАхжЮбтГљУхёТ║љт«їТЋ┤ТђДТаАжфїУ»ду╗єТїЄтЇЌ (opens new window)ТюЅТХЅтЈітѕ░т░єТќЄС╗ХУ┐ЏУАїsha384тіат»єтљјbase64у╝ќуаЂТЮЦТБђжфїТќЄС╗ХТў»тљдУбФу»АТћ╣сђѓт»╣тЁХСИГућетѕ░уџётиЦтЁиopensslТ»ћУЙЃТёЪтЁ┤УХБ№╝їТЅђС╗ЦжА║тИдС║єУДБС║єтіат»єу«ЌТ│Ћ№╝їу«ЌТў»тіат»єу«ЌТ│ЋуџёСИђС║ЏтИИУ»єсђѓ

      С╗ЦСИІСИ║ТГБТќЄсђѓ

      тіат»єу«ЌТ│ЋСИђУѕгтЈ»тѕєСИ║СИцуДЇ№╝їтЇ│тЇЋтљЉтіат»єу«ЌТ│ЋтњїтЈїтљЉтіат»єу«ЌТ│Ћсђѓ

      тЈїтљЉтіат»єТў»тіат»єу«ЌТ│ЋСИГТюђтИИућеуџё№╝їт«Ѓт░єтЈ»С╗ЦуЏ┤ТјЦуљєУДБуџёТўјТќЄТЋ░ТЇ«тіат»єСИ║СИЇтЈ»уЏ┤ТјЦуљєУДБуџёт»єТќЄТЋ░ТЇ«№╝їтюежюђУдЂуџёТЌХтђЎ№╝їтЈѕтЈ»С╗ЦСй┐ућеСИђт«џуџёу«ЌТ│Ћт░єУ┐ЎС║Џтіат»єС╗Цтљјуџёт»єТќЄТЋ░ТЇ«УДБт»єСИ║тјЪТЮЦуџёТўјТќЄТЋ░ТЇ«сђѓтЈїтљЉтіат»єжђѓтљѕС║јжџљуДўжђџУ«»№╝їТ»ћтдѓ№╝їТ»ћтдѓтюеуйЉу╗юСИіТ│етєїућеТѕиТѕќУђЁУ┤ГС╣░тЋєтЊЂТЌХ№╝їТЈљС║цуџёуюЪт«ътДЊтљЇсђЂУ║ФС╗йУ»ЂтЈиуаЂсђЂжЊХУАїУ┤дтЈисђЂС┐АућетЇАт»єуаЂуГЅС┐АТЂ»№╝їт║ћтйЊжђџУ┐ЄтЈїтљЉтіат»єу«ЌТ│Ћтіат»єС╗Цтљј№╝їтєЇтюеуйЉу╗юСИіС╝аУЙЊ№╝їУ┐ЎТаи№╝їтЈ»С╗ЦТюЅТЋѕуџёжў▓ТГбж╗Љт«буџёРђютЂитљгРђЮ№╝їС┐ЮУ»ЂТЋ░ТЇ«уџёт«ЅтЁе№╝їтљїТЌХ№╝їуйЉуФЎТјЦТћХтѕ░ТѕЉС╗гуџёТЋ░ТЇ«С╗Цтљј№╝їтЈ»С╗ЦжђџУ┐ЄУДБт»єу«ЌТ│ЋТЮЦУјитЙЌтЄєуА«уџёС┐АТЂ»сђѓ

      тЈїтљЉтіат»єТЌбтЈ»С╗Цтіат»є№╝їтЈѕтЈ»С╗ЦУДБт»єсђѓУђїтЇЋтљЉтіат»єтѕЎтѕџтЦйуЏИтЈЇ№╝їт«ЃтЈфУЃйт»╣ТўјТќЄТЋ░ТЇ«У┐ЏУАїтіат»є№╝їУђїСИЇУЃйт»╣тіат»єС║єуџёт»єТќЄТЋ░ТЇ«тєЇУДБт»єТѕљтјЪТЮЦуџёТўјТќЄТЋ░ТЇ«сђѓтЈ»УЃйУ»╗УђЁС╝џУ«цСИ║№╝їСИЇУЃйУДБт»єуџётіат»єу«ЌТ│ЋТюЅС╗ђС╣ѕСйюућетЉб№╝Ътюет«ъжЎЁт║ћућеСИГ№╝їт»╣Уй»С╗Ху│╗у╗ЪТЋ░ТЇ«т║ЊСИГуџёу│╗у╗ЪућеТѕиС┐АТЂ»№╝ѕтдѓућеТѕит»єуаЂ№╝Ѕтіат»є№╝їт░▒Тў»СИђСИфтЁИтъІуџёСЙІтГљсђѓтйЊућеТѕиТ│етєїСИђСИфТќ░уџёУ┤дтЈиТЌХ№╝їтЁХућеТѕит»єуаЂС┐АТЂ»СИЇТў»уЏ┤ТјЦС┐ЮтГўтѕ░ТЋ░ТЇ«т║Њ№╝їУђїТў»у╗ЈУ┐ЄтЇЋтљЉтіат»єтљјтєЇС┐ЮтГў№╝їУ┐ЎТаи№╝їтЇ│Сй┐У┐ЎС║ЏУ┤дтЈиС┐АТЂ»УбФТ│ёжю▓№╝їтѕФС║║С╣ЪСИЇУЃйтЙЌтѕ░уЏИт║ћуџёућеТѕит»єуаЂ№╝їтйЊуёХС╣Ът░▒УЙЙСИЇтѕ░уЏЌуфЃУ┤дтЈиуџёуЏ«уџёсђѓ

      С╣ЪУ«И№╝їТюЅуџёУ»╗УђЁС╝џТЃ│№╝їТЌбуёХтЇЋтљЉтіат»єТЌаТ│ЋУДБт»є№╝їжѓБућеТѕиСИђТЌдт┐ўУ«░С║єт»єуаЂ№╝їтЄаС╣јТ▓АТюЅтіъТ│ЋТЅЙтЏътјЪТЮЦуџёт»єуаЂ№╝їтЈѕУ»ЦТђјС╣ѕтіътЉб№╝ЪУ┐ЎуА«т«ъТў»СИђСИфТ»ћУЙЃж║╗уЃдуџёжЌ«жбў№╝їтдѓТъюТў»СИђСИфтіЪУЃйу«ђтЇЋуџёУй»С╗Ху│╗у╗Ъ№╝їС╣ЪУ«ИуюЪуџётЈфУЃйтѕажЎцт┐ўУ«░т»єуаЂуџёУ┤дтЈиС┐АТЂ»№╝їуёХтљјтєЇжЄЇТќ░Т│етєїТЮЦУДБтє│сђѓСйєТў»Т»ћУЙЃт«їтќёуџёУй»С╗Ху│╗у╗Ъ№╝їСИђУѕгжЃйтЈ»С╗ЦУ«ЕућеТѕиТЈљСЙЏСИђС║ЏтЁХС╗ќуџёТ│етєїС┐АТЂ»№╝їтдѓтЄ║ућЪтю░тЮђсђЂуѕХТ»Їтд╗тГљуџёУЂїСИџуГЅ№╝їтдѓТъюТГБуА«№╝їтѕЎуЏ┤ТјЦу╗ЎућеТѕиТЈљСЙЏС┐«Тћ╣т»єуаЂуџётіЪУЃй№╝їуёХтљјућеТќ░т»єуаЂТЏ┐ТЇбТјЅТЋ░ТЇ«т║ЊСИГуџёТЌДт»єуаЂ№╝їУ┐ЎТаи№╝їућеТѕитЈфУдЂУ«░СйЈТќ░уџёућеТѕит»єуаЂт░▒тЈ»С╗ЦС║єсђѓ

      # тЇЋтљЉтіат»є

      тИИућеуџётЇЋтљЉтіат»єТюЅ:

      • MD5№╝џMD5СИЦТа╝ТёЈС╣ЅСИіТЮЦУ»┤№╝їСИЇТў»у╝ќуаЂС╣ЪСИЇТў»тіат»є№╝їУђїТў»ТЉўУдЂу«ЌТ│Ћ№╝їС╣ЪтЈФтЂџтЊѕтИїу«ЌТ│ЋтњїТЋБтѕЌу«ЌТ│Ћ№╝їт«ЃуџётЁИтъІт║ћућеТў»№╝џжў▓ТГбу»АТћ╣тњїТаАжфїТЋ░ТЇ«сђѓТЌаУ«║Тў»тцџжЋ┐уџёУЙЊтЁЦ№╝їMD5жЃйС╝џУЙЊтЄ║жЋ┐т║дСИ║128bitsуџёСИђСИфСИ▓ (жђџтИИуће16У┐ЏтѕХУАеуц║СИ║32СИфтГЌугд)сђѓжюђУдЂТ│еТёЈуџёТў»№╝џТЉўУдЂу«ЌТ│ЋТў»СИЇтЈ»С╗ЦжђєуџёсђѓжђџУ┐Єтіат»єтљјуџёТЋ░ТЇ«Тў»СИЇУЃйтЙЌтѕ░тјЪтДІТЋ░ТЇ«уџёсђѓ
      • SHA: Secure Hash Algorithm№╝їт«ЅтЁеТЋБтѕЌу«ЌТ│Ћ№╝їТЋ░тГЌуГЙтљЇуГЅт»єуаЂтГдт║ћућеСИГжЄЇУдЂуџётиЦтЁи№╝їУбФт╣┐Т│Џтю░т║ћућеС║јућхтГљтЋєтіАуГЅС┐АТЂ»т«ЅтЁежбєтЪЪсђѓУЎйуёХSHAСИјMD5жђџУ┐Єуб░ТњъТ│ЋжЃйУбФуа┤УДБС║є№╝їСйєТў»SHAС╗ЇуёХТў»тЁгУ«цуџёт«ЅтЁетіат»єу«ЌТ│Ћ№╝їУЙЃС╣ІMD5ТЏ┤СИ║т«ЅтЁесђѓ
      • HMAC№╝џHash Message Authentication Code№╝їТЋБтѕЌТХѕТЂ»жЅ┤тѕФуаЂ№╝їтЪ║С║јт»єжњЦуџёHashу«ЌТ│ЋуџёУ«цУ»ЂтЇЈУ««сђѓТХѕТЂ»жЅ┤тѕФуаЂт«ъуј░жЅ┤тѕФуџётјЪуљєТў»№╝їућетЁгт╝ђтЄйТЋ░тњїт»єжњЦС║ДућЪСИђСИфтЏ║т«џжЋ┐т║дуџётђ╝СйюСИ║У«цУ»ЂТаЄУ»є№╝їућеУ┐ЎСИфТаЄУ»єжЅ┤тѕФТХѕТЂ»уџёт«їТЋ┤ТђДсђѓСй┐ућеСИђСИфт»єжњЦућЪТѕљСИђСИфтЏ║т«џтцДт░Јуџёт░ЈТЋ░ТЇ«тЮЌ№╝їтЇ│MAC№╝їт╣Хт░єтЁХтіатЁЦтѕ░ТХѕТЂ»СИГ№╝їуёХтљјС╝аУЙЊсђѓТјЦТћХТќ╣тѕЕућеСИјтЈЉжђЂТќ╣тЁ▒С║Фуџёт»єжњЦУ┐ЏУАїжЅ┤тѕФУ«цУ»ЂуГЅсђѓ

      тЇЋтљЉтіат»єу«ЌТ│ЋуЅ╣тЙЂ№╝џ

      • УЙЊтЁЦСИђТаи№╝їУЙЊтЄ║т┐ЁуёХуЏИтљї
      • жЏфт┤ЕТЋѕт║ћ№╝џУЙЊтЁЦуџётЙ«т░ЈТћ╣тЈў№╝їт░єт╝ЋУхиу╗ЊТъютиетцДуџётЈўтїќ
      • т«џжЋ┐УЙЊтЄ║№╝їТЌаУ«║тјЪтДІТЋ░ТЇ«тцџтцД№╝їу╗ЊТъютцДт░ЈжЃйТў»уЏИтљїуџё
      • СИЇтЈ»жђє№╝їТЌаТ│ЋТа╣ТЇ«уЅ╣тЙЂуаЂУ┐ўтјЪтјЪТЮЦуџёТЋ░ТЇ«

      тіат»єтиЦтЁи№╝џmd5sumсђЂsha1sumсђЂopenssl dgst

      # тЈїтљЉтіат»є

      тЈїтљЉтіат»єтЈѕтѕєт»╣уД░тіат»єтњїжЮът»╣уД░тіат»єсђѓ

      # т»╣уД░тіат»є

      жЄЄућетЇЋжњЦт»єуаЂу│╗у╗Ъуџётіат»єТќ╣Т│Ћ№╝їтљїСИђСИфт»єжњЦтЈ»С╗ЦтљїТЌХућеСйюС┐АТЂ»уџётіат»єтњїУДБт»є№╝їУ┐ЎуДЇтіат»єТќ╣Т│ЋуД░СИ║т»╣уД░тіат»є№╝їС╣ЪуД░СИ║тЇЋт»єжњЦтіат»єсђѓ

      • тИИућеуџёт»╣уД░тіат»єТюЅ№╝џ

        • DES(Data Encryption Standard)№╝џТЋ░ТЇ«тіат»єТаЄтЄє№╝їжђЪт║дУЙЃт┐Ф№╝їжђѓућеС║јтіат»єтцДжЄЈТЋ░ТЇ«уџётю║тљѕ
        • IDEA(тЏйжЎЁТЋ░ТЇ«тіат»єу«ЌТ│Ћ)
        • 3DES№╝џТў»тЪ║С║јDES№╝їт»╣СИђтЮЌТЋ░ТЇ«ућеСИЅСИфСИЇтљїуџёт»єжњЦУ┐ЏУАїСИЅТгАтіат»є№╝їт╝║т║дТЏ┤жФў
        • AES :№╝ѕAdvanced Encryption Standard№╝Ѕ№╝џжФўу║Дтіат»єТаЄтЄє№╝їТў»СИІСИђС╗Буџётіат»єу«ЌТ│ЋТаЄтЄє№╝їжђЪт║дт┐Ф№╝їт«ЅтЁеу║ДтѕФжФў

      • т»╣уД░тіат»єу«ЌТ│ЋуЅ╣тЙЂ№╝џ

        • тіат»єТќ╣тњїУДБт»єТќ╣Сй┐ућетљїСИђСИфт»єжњЦ
        • тіат»єУДБт»єуџёжђЪт║дТ»ћУЙЃт┐Ф№╝їжђѓтљѕТЋ░ТЇ«Т»ћУЙЃжЋ┐ТЌХуџёСй┐уће
        • т»єжњЦС╝аУЙЊуџёУ┐ЄуеІСИЇт«ЅтЁе№╝їСИћт«╣ТўЊУбФуа┤УДБ

      # жЮът»╣уД░тіат»є

      • тИИУДЂуџёжЮът»╣уД░тіат»єу«ЌТ│ЋТюЅ

        • RSA№╝џRSAТў»уггСИђСИфТЌбУЃйућеС║јТЋ░ТЇ«тіат»єС╣ЪУЃйућеС║јТЋ░тГЌуГЙтљЇуџёу«ЌТ│Ћсђѓт«ЃТўЊС║јуљєУДБтњїТЊЇСйю№╝їС╣ЪтЙѕТхЂУАїсђѓу«ЌТ│ЋуџётљЇтГЌС╗ЦтЈЉТўјУђЁуџётљЇтГЌтЉйтљЇ№╝џRon Rivest, Adi Shamir тњїLeonard Adleman

        тљју╗ГС╝џуюІСИђСИІRASуџётіат»єтјЪуљє

        • ECC№╝ѕуД╗тіеУ«ЙтцЄуће№╝Ѕ№╝џТцГтюєТЏ▓у║┐т»єуаЂу╝ќуаЂтГд
        • El Gamal (opens new window)№╝џElGamalтіат»єу«ЌТ│ЋТў»СИђСИфтЪ║С║јУ┐фУЈ▓-УхФт░ћТЏ╝т»єжњЦС║цТЇб(DiffieРђЊHellman key exchange) (opens new window)уџёжЮът»╣уД░тіат»є (opens new window)у«ЌТ│Ћ
        • DSA№╝ѕТЋ░тГЌуГЙтљЇуће№╝Ѕ№╝џТЋ░тГЌуГЙтљЇу«ЌТ│Ћ№╝їТў»СИђуДЇТаЄтЄєуџё DSS№╝ѕТЋ░тГЌуГЙтљЇТаЄтЄє№╝Ѕ

      уггтЁГт▒ітЏйжЎЁт»єуаЂтГдС╝џУ««т»╣т║ћућеС║јтЁгжњЦт»єуаЂу│╗у╗Ъуџётіат»єу«ЌТ│ЋТјеУЇљС║єСИцуДЇ№╝џтЪ║С║јтцДТЋ┤ТЋ░тЏатГљтѕєУДБжЌ«жбў№╝ѕIFP№╝ЅуџёRSAу«ЌТ│ЋтњїтЪ║С║јТцГтюєТЏ▓у║┐СИіуд╗ТЋБт»╣ТЋ░У«Ау«ЌжЌ«жбў№╝ѕECDLP№╝ЅуџёECCу«ЌТ│Ћсђѓ
